## Instalasi Project

Clone project

```bash
  git clone https://github.com/dodfrm/Sobat.git
```

Pindah ke directory project

```bash
  cd Sobat
```

Install dependencies

```bash
  npm install
```

Jalankan server

```bash
  npm run dev
```
Website akan berjalan pada http://localhost:5173
